Multi-comparison template for Illumina methylation arrays.

Scope:
- global all-sample overview (combined inputs)
- multi-comparison DMP
- per-comparison DMR scaffold
- per-comparison enrichment
- drilldown exports

Input expectation:
- one or more upstream process runs in `config/input_registry.csv`
- each run points to a `results/rds` directory containing one of:
  `adjustedset.rds`, `filteredset.rds`, `normset.rds`, or `rgset.rds`
- comparisons defined in `config/comparisons.csv`

## Workflow

1. Run `illumina_methylation_process` templates first.
2. Render this template.
3. (Default) post-render hook auto-populates `config/input_registry.csv` from active process templates in `project.yaml`.
4. Review `config/input_registry.csv` and set `enabled`/filters.
5. Optionally harmonize labels in `config/group_map.csv`.
6. Run `pixi run sync-samples` (or use `./run.sh`, which syncs automatically by default).
7. Define enabled comparisons in `config/comparisons.csv`.
8. Run `./run.sh`.

## Key Files

- `config/input_registry.csv`:
  - `run_id,dataset_id,process_template,array_type,genome_build,processed_results_dir,samples_file,enabled,include_samples,exclude_samples`
- `config/group_map.csv`:
  - harmonize raw groups into analysis groups
- `config/comparisons.csv`:
  - `comparison_id,name,base_group,target_group,enabled,covariates,alpha,delta_beta_min`
- `config/samples.csv`:
  - merged sample table generated by `sync-samples`

Notes:
- probes are merged by intersection across enabled runs
- `sample_id` must be globally unique across enabled runs
- one run executes all enabled comparisons from `comparisons.csv`

## Render

```bash
bpm template render illumina_methylation_compare \
  --param array_type=AUTO \
  --param group_col=group \
  --dir /path/to/project
```

`study_name` is optional; default is the rendered template id.

Disable auto-discovery of input registry:

```bash
bpm template render illumina_methylation_compare \
  --param auto_discover_inputs=false \
  --dir /path/to/project
```

## Run

```bash
bpm template run illumina_methylation_compare --dir /path/to/project
# or inside rendered folder
./run.sh
```

## Outputs

- `reports/00_index.html`: global overview + all-sample PCA
- `reports/01_dmp.html`: DMP summary across enabled comparisons
- `reports/02_dmr.html`: per-comparison DMR scaffold summary
- `reports/03_enrichment.html`: per-comparison GO/KEGG summary
- `reports/04_drilldown.html`: drilldown export summary
- `results/tables/comparisons/<comparison_id>/...`: per-comparison tables
- `results/figures/comparisons/<comparison_id>/...`: per-comparison figures
